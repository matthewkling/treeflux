This repo contains source code for the `FIA Climate Analogs` Shiny app.

To run the app: from RStudio, select `New Project` -\> `From version control` -\> `Git`. Paste `https://github.com/matthewkling/fia_analogs.git` into the `Repository URL` field, choose where to save the project, and click `Create project`. Then open the file called `app.R` and click the `run app` button.\
\
This app requires the [`analogs` R package](https://matthewkling.github.io/analogs/). You can install it via `devtools::install_github("matthewkling/analogs")`.
